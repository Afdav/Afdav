# 👨‍💻 Edgar AF – Portfolio de Proyectos.

Desarrollador Full Stack especializado en soluciones empresariales, SaaS y sistemas de gestión.

Este repositorio presenta una colección de proyectos reales desarrollados para clientes y organizaciones, enfocados en resolver problemas del mundo real en áreas como:

- Facturación electrónica (SIFEN – Paraguay)
- Sistemas comerciales (farmacia, agropecuario)
- Plataformas SaaS multi-tenant
- Sistemas de gestión y seguimiento de proyectos

> ⚠️ Nota: Los proyectos son privados por razones comerciales, pero aquí se documentan sus funcionalidades, arquitectura y evidencias visuales reales.

---

## 🚀 Tecnologías principales

- Backend: Laravel (8, 10), PHP (7.4 – 8.3)
- Frontend: Blade, Bootstrap 5.x
- Bases de datos: MySQL, PostgreSQL
- Arquitectura: SaaS multi-tenant, APIs REST
- Integraciones: SIFEN (facturación electrónica Paraguay)

---

## 📂 Proyectos destacados

### 🧾 [FactuTeeté – Facturación Electrónica (SaaS)](./factuteete/README.md)
Sistema multiempresa integrado con SIFEN, con API REST para terceros.

### 💊 [Kuatiañemu FarmaSoft – Sistema de Farmacia](./kuatianemu-farmasoft/README.md)
Gestión completa de farmacia con control de vencimientos, créditos y cajas.

### 🌱 [Kuatiañemu AgroSoft – Sistema Agropecuario](./kuatianemu-agrosoft/README.md)
Sistema comercial adaptado al sector agro con múltiples esquemas de precios.

### 📋 [Backlog System – Gestión de Tickets](./backlog-system/README.md)
Sistema de seguimiento de tareas, proyectos y prioridades empresariales.
